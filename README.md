# AI Grader Project

Đây là project đồ án xây dựng hệ thống hỗ trợ quản lý bài tập lập trình và chấm điểm tự động.

## Công nghệ sử dụng

* Python
* Flask
* SQLite
* HTML/CSS
* Mô hình MVC

## Chức năng chính

* Đăng nhập và phân quyền người dùng
* Quản lý sinh viên
* Quản lý bài tập lập trình
* Quản lý tài liệu học tập
* Sinh và quản lý bài trắc nghiệm
* Chấm điểm bài làm
* Hiển thị kết quả chấm điểm
* Xuất bảng điểm

## Cấu trúc project

```text
ai_grader/
├── app.py
├── models.py
├── setup_db.py
├── templates/
├── static/
└── test_api.py
```

## Mục tiêu học tập

Project này được sử dụng để luyện lại các kỹ năng:

* Git/GitHub
* Flask route
* MVC trong Flask
* SQLite database
* Xử lý form
* CRUD cơ bản
* Đọc và sửa code
* Viết lại từng module bằng tay
